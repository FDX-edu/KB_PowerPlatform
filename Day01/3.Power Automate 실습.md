# 3. Power Automate 실습

네이버 API를 활용하여 뉴스 기사를 수집하고 요약하여 Teams 메시지로 전달하는 자동화를 만들어 봅시다.


## 네이버 API 설정

네이버 API를 호출하기 위해선 [네이버 개발자 센터](https://developers.naver.com/)에 등록이 필요합니다.  
애플리케이션을 생성하면 Client id와 Client Secret을 발급받을 수 있습니다.
